# Backend app "Todo-List" 

# About project

This is a small backend application "Todo-List" allows adding tasks.
All tasks are stored in the database and linked to the user with id, also tasks have statuses
"todo", "in progress", "done". The user can change the status of the task, delete it or add a new one.
All tasks have a deadline date and user gets notification on telegram about ending of the deadline.

# Technologies
- Java
- Spring Boot
- PosgreSQL
- Telegram Bot API
